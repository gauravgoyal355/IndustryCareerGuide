# IndustryCareerGuide.com - Next.js Frontend\n\nA comprehensive career guidance platform for STEM PhDs transitioning to industry, built with Next.js and optimized for Vercel deployment.\n\n## 🚀 Quick Start\n\n### Prerequisites\n- Node.js 18+ \n- npm or yarn\n- Git\n\n### Local Development\n\n1. **Install dependencies:**\n   ```bash\n   npm install\n   ```\n\n2. **Run development server:**\n   ```bash\n   npm run dev\n   ```\n\n3. **Open your browser:**\n   Navigate to `http://localhost:3000`\n\n## 📦 Project Structure\n\n```\nIndustryCareerGuide/\n├── components/           # Reusable React components\n│   ├── Layout.js        # Main layout wrapper\n│   ├── Header.js        # Navigation header\n│   ├── Footer.js        # Site footer\n│   ├── CareerMap.js     # Interactive career visualization\n│   └── ActionPlan.js    # Personalized action plans\n├── data/                # Static JSON data files\n│   ├── career_assessment_quiz.json\n│   ├── careerPaths.json\n│   └── careerTrajectories.json\n├── pages/               # Next.js pages\n│   ├── api/            # Serverless API routes\n│   │   ├── matchCareer.js\n│   │   └── actionPlan.js\n│   ├── quiz/index.js   # Career assessment\n│   ├── results/index.js # Quiz results\n│   ├── careerMap/index.js # Career visualizations\n│   ├── actionPlan/index.js # Action plan generator\n│   └── index.js        # Landing page\n├── styles/\n│   └── globals.css     # Global styles with Tailwind\n└── public/             # Static assets\n```\n\n## 🔧 Key Features\n\n### Static Optimization\n- **Minimal serverless usage** - Only 2 API routes for dynamic features\n- **JSON-based data** - All static content served from JSON files\n- **Client-side routing** - Fast navigation without page reloads\n- **Optimized for Vercel** - Zero-config deployment\n\n### Interactive Components\n- **Career Assessment Quiz** - 23-question evaluation\n- **Dynamic Career Matching** - AI-powered career recommendations\n- **Interactive Career Maps** - Visual progression paths\n- **Personalized Action Plans** - Custom roadmaps with courses and certifications\n\n### Performance Features\n- **Static generation** - Fast loading times\n- **Responsive design** - Mobile-first approach\n- **SEO optimized** - Meta tags, structured data\n- **Accessibility** - WCAG compliant components\n\n## 🚀 Deployment to Vercel\n\n### Method 1: GitHub Integration (Recommended)\n\n1. **Push to GitHub:**\n   ```bash\n   git init\n   git add .\n   git commit -m \"Initial commit - IndustryCareerGuide frontend\"\n   git branch -M main\n   git remote add origin https://github.com/YOUR_USERNAME/IndustryCareerGuide.git\n   git push -u origin main\n   ```\n\n2. **Deploy on Vercel:**\n   - Visit [vercel.com](https://vercel.com)\n   - Click \"New Project\"\n   - Import your GitHub repository\n   - Vercel will auto-detect Next.js and deploy\n   - Your site will be live at `https://your-project.vercel.app`\n\n### Method 2: Vercel CLI\n\n1. **Install Vercel CLI:**\n   ```bash\n   npm install -g vercel\n   ```\n\n2. **Deploy:**\n   ```bash\n   vercel\n   ```\n\n3. **Follow prompts:**\n   - Link to existing project or create new\n   - Confirm settings\n   - Deploy automatically\n\n## ⚙️ Environment Configuration\n\n### Production Settings\n\nVercel will automatically configure:\n- **Build Command:** `npm run build`\n- **Output Directory:** `.next`\n- **Install Command:** `npm install`\n- **Development Command:** `npm run dev`\n\n### Custom Domain (Optional)\n\n1. **Add domain in Vercel dashboard**\n2. **Update DNS records** as instructed\n3. **SSL automatically configured**\n\n## 🔄 Continuous Deployment\n\n**Automatic deployments** trigger on:\n- Push to `main` branch → Production deployment\n- Push to other branches → Preview deployments\n- Pull requests → Preview deployments with unique URLs\n\n### Manual Deployment Commands\n\n```bash\n# Deploy to production\nvercel --prod\n\n# Deploy preview\nvercel\n\n# Deploy specific branch\nvercel --target production\n```\n\n## 📊 Performance Optimization\n\n### Vercel-Specific Optimizations\n\n- **Static file caching** -
